PENGUIN DASH 0.1.0
==================

1. Extract the complete ZIP to a writable folder.
2. Run PenguinDash.exe. Python is not required.
3. Local saves and logs are stored under %LOCALAPPDATA%\PenguinDash\.

Diagnostic smoke test:
  PenguinDash.exe --self-test

If startup fails, include the JSON file shown on the safe error screen when
reporting the problem. Known limitations are listed in KNOWN_ISSUES.md.
